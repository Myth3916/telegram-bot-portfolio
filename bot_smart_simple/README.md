# 🤖 Умный бот с базой знаний

> Telegram-бот с настраиваемой базой ответов. Идеально для FAQ, поддержки и первого контакта с клиентами.  
> **Цена:** 3000₽ | **Срок:** 2-3 дня

## ✨ Возможности

- 💬 **База знаний** — ответы на частые вопросы
- ➕ **Легко расширять** — команда `/add` для добавления ответов
- 💾 **Сохранение** — база хранится в JSON-файле
- 🎲 **Умные ответы** — если не нашёл точный ответ, даёт случайный
- 📋 **Просмотр базы** — команда `/list` показывает все записи
```
## 🚀 Быстрый запуск

### 1. Клонирование
```bash
git clone https://github.com/Myth3916/bot_smart_simple.git
cd bot_smart_simple
```

### 2. Установка
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 3. Настройка
Создай `.env`:
```env
BOT_TOKEN=твой_токен
```

### 4. Запуск
```bash
python main.py
```

## 📚 Команды

- `/start` — приветствие
- `/help` — справка
- `/add ключ|ответ` — добавить ответ в базу
- `/list` — показать все записи

## 💡 Примеры использования

**Добавить ответ:**
```
/add цена|Наши цены от 1500₽. Подробности: @ConversAIonist
```

**Проверить:**
```
цена → Наши цены от 1500₽. Подробности: @ConversAIonist
```
