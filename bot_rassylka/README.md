# 📢 Бот для рассылки с админ-панелью

> Профессиональный инструмент для управления базой подписчиков и безопасных рассылок.  
> Цена проекта: **4500₽** | Технологии: Python, SQLite, pyTelegramBotAPI
```
## ✨ Возможности

- 💾 **Автоматическая база данных**: каждый нажавший `/start` сохраняется в SQLite.
-  **Защита от админов**: команды управления доступны только владельцу бота.
- 📊 **Статистика**: мгновенный подсчет активных подписчиков (`/stats`).
- 📢 **Безопасная рассылка**:
  - Автоматическая задержка между сообщениями (защита от бана Telegram).
  - Обработка ошибок (бот не падает, если пользователь заблокировал рассылку).
  - Отчет о доставке (успешно / ошибки).

## 🚀 Быстрый запуск

### 1. Клонирование
```bash
git clone https://github.com/Myth3916/bot_rassylka.git
cd bot_rassylka
```

### 2. Установка зависимостей
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 3. Настройка
Создай файл `.env`:
```env
BOT_TOKEN=твой_токен
ADMIN_ID=твой_id
```

### 4. Запуск
```bash
python main.py
```

## 🛡️ Безопасность
Код включает встроенную защиту от превышения лимитов Telegram API (`time.sleep`) и обработку исключений, чтобы бот оставался онлайн даже при массовых блокировках пользователей.
