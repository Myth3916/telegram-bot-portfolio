# 📩 Lead Bot — Telegram-бот для сбора заявок

Простой и надёжный бот для малого бизнеса. Собирает контакты клиентов, сохраняет в базу и позволяет выгружать в Excel.

## 🚀 Функции

- ✅ Сбор заявок (имя + телефон)
- ✅ Сохранение в SQLite базу данных
- ✅ Админ-панель со статистикой
- ✅ Экспорт заявок в CSV (Excel)
- ✅ Защита админки по chat_id
- ✅ Работа через прокси

## 🛠 Установка

### 1. Установи зависимости
```bash
pip install -r requirements.txt
```
